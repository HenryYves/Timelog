设置

按 T 键快速创建时默认时间块长度
例：timelog-backup-2026-07-04-1730.json
输入绝对路径，留空则使用 AppData
启用后隐藏原生标题栏，顶部右侧显示窗口控制按钮
关闭后不再自动写入备份文件，避免数据量大时影响性能；你仍可随时点「立即备份」手动保存。
每天首次操作时，会把当前数据快照存为上一次日期的备份（timelog-backup-日期.json）；最多保留 4 个，超出自动删除最早的。
0 = 保留全部；设 7 则每天首次操作时自动删掉第 8 个及更早有数据的天，仅保留最近 7 个有数据的天。

操作指南

时间块操作

  • 在时间轴上拖动选择时段，松开后填写内容
  • 左键点击色块编辑
  • 右键勾选/取消色块（多选）
  • 拖动色块顶部/底部边缘调整时长

快捷键

  • Esc 关闭弹窗 / 取消选择
  • Delete 删除选中的时间块
  • Ctrl + C 复制选中
  • Ctrl + V 粘贴到鼠标悬停位置
  • T 快速记录当前时间
  • ? 打开此帮助
  • ↑↓ 拖动时微调时间（±1分钟）

数据备份

  • 桌面应用自动静默备份到 timelog_data 文件夹
  • 浏览器中请用「导出备份」按钮手动保存 JSON
  • 每天首次操作时生成上一日期的备份，最多保留 4 个

文本导入/导出

  • 用「导出文本」复制当天内容，可粘贴到笔记
  • 用「文本导入」把之前导出的文本解析回来
  • 也支持「导出备份」→「导入」做完整 JSON 迁移
