## v0.8.8

### Added
- 饼图 SVG 重写：引导线标注 + 扇区 hover 交互 + 标签防重叠
- 图表交互属性：开启后 hover 扇区/柱子在卡片头部显示数据
- 视图自定义名称
- 导出/导入按内容选择：数据、标签、统计视图、设置（JSON v3）
- 自动备份纳入统计视图和设置

### Changed
- 饼图从 CSS `conic-gradient` 替换为 SVG + 引导线
- 统计面板视觉美化：卡片装饰条、柱状图渐变、筛选 pill、动画
- 日期输入改用 text 类型 + 格式校验提示
- 自定义日期范围持久化记忆
- 移除展开视图和日期分隔符

### Fixed
- 柱状图 `<span>` 无 `display:block` 导致高度 0 不可见
- 饼图大扇区（>180°）引导线方向计算错误
- 颜色 fallback 校验加固 + 导入设置后 store 刷新
- 属性面板焦点管理 + 固定高度滚轮 + checkbox 焦点可见
