# 统计面板 Spec

> 仪表盘式统计视图——多个可配置图表，按标签统计时长。

## 整体布局

```
┌─────────────────────────────────────────┐
│ 统计                      [时间范围 ▾] │  ← 下拉栏，同 More 菜单风格
├─────────────────────────────────────────┤
│                                         │
│ ┌───────────────────────────────────┐   │
│ │ 饼图：标签时长分布        [⚙]    │   │  ← 大卡片，纵向排列
│ │                                   │   │
│ └───────────────────────────────────┘   │
│ ┌───────────────────────────────────┐   │
│ │ 柱状图：每日趋势            [⚙]  │   │
│ │                                   │   │
│ └───────────────────────────────────┘   │
│                                         │
│ [+ 添加视图]                             │
└─────────────────────────────────────────┘
```

- 卡片纵向排列，占满宽度，支持滚动
- 后续可加自定义布局（拖拽调整卡片大小/位置），第一版先纵向堆叠

## 全局时间过滤器

下拉栏按钮切换，默认**上次离开面板时的时间段**。如果上次选了"今天"且已过天，打开仍显示"今天"（当天数据）。

| 选项 | 含义 |
|------|------|
| 今天 | 当日 0:00 ~ 当前时间 |
| 24h | 过去 24 小时 |
| 本周 | 周一 0:00 ~ 当前时间 |
| 24×7h | 过去 168 小时 |
| 最近 7 天 | 过去 7 个自然日 |
| 本月 | 月初 ~ 当前时间 |
| 自定义 | 起止日期选择 |

## 添加视图流程

点击 `[+ 添加视图]` → 弹出创建弹窗，配置以下属性：

- **图表类型**：饼图 / 柱状图
- **[ ] 只记录第一个标签**（多标签块是否每个标签都算时长）
- **[ ] 显示图例**
- **[ ] 图例显示数据**（标签名 + 分钟数）
- **[ ] 图例显示占比**（标签名 + 百分比）
- **[ ] 图中显示数据**
- **[ ] 图中显示占比**

确认后创建卡片。卡片用 `sortOrder` 排序，新卡片加到最后。

## 卡片设置（⚙ 按钮）

右上角齿轮弹出 mini 设置层，可修改所有创建时配置的属性。另加：

- **删除此视图**（红色按钮）

## 图表卡片数据结构

```javascript
cards: [
  {
    id: string,
    type: 'pie' | 'bar',
    onlyFirstTag: boolean,
    showLegend: boolean,
    legendData: boolean,
    legendPercent: boolean,
    chartData: boolean,
    chartPercent: boolean,
    sortOrder: number,
  }
]
```

持久化到 `localStorage` key `timelog:stats-cards`。

## 数据计算

统计面板内 computed 按全局时间范围遍历所有天 → 所有时间块：

1. 如果 `onlyFirstTag` → 只取 `block.tags[0]`
2. 否则 → 每个 tag 都计 `duration` 分钟
3. 按标签名聚合 `duration`（分钟）
4. 所有图表共用同一份聚合数据

## 图表实现

纯 CSS/SVG 手绘，不引入图表库。

- 柱状图：横向或纵向 bar，宽度/高度按分钟数比例
- 饼图：`conic-gradient()` 实现扇区
- 颜色使用 `tagStore.colorOf(name).hex`
- 单位显示："Xh Ym" 或 "X 分钟"

## 空状态

无数据 → 卡片内显示灰色文字"暂无数据"。

无图表 → 面板中间显示灰色文字"点击 [+ 添加视图] 创建第一个图表"。

## 入口

- `App.vue` 头部 More 下拉菜单，放在 `标签` 和 `文本导入` 之间
- 图标：复制统计 SVG 到 `frontend/src/assets/stats.svg`（源文件 `D:\a_my\seldom\computer_system\transfer\SP_xx_zd\统计.svg`）
- 可选快捷键：`S`

## 组件结构

- `StatsPanel.vue` — 主面板：时间筛选 + 卡片列表 + 添加按钮
- 卡片内联渲染（不单独拆组件）
- 创建/设置弹窗用 `v-if` 浮层

## 持久化

| key | 内容 |
|-----|------|
| `timelog:stats-cards` | 卡片配置数组 |
| `timelog:stats-time-range` | 上次时间范围选择 |
